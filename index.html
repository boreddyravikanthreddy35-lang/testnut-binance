Capabilities
Built for Production
Every component is clean, reusable, and tested against the live testnet.
⚡
01 — Order Execution

3 Order Types

MARKET fills at best price instantly. LIMIT rests on the book with GTC time-in-force. LIMIT_IOC (bonus) fills what it can immediately and cancels the rest — all via a single CLI command.

🔒
02 — Security

HMAC-SHA256 Authentication

Every API request is signed with your secret key using HMAC-SHA256. Server-time is synced on startup to auto-correct clock-skew errors. Credentials loaded from .env — never hardcoded.

📝
03 — Observability

Structured Rotating Logs

Every API request, response body, order ID, and error is logged at DEBUG level to a rotating 5 MB file. Console shows only WARNING and above so the terminal stays clean.

✓
04 — Robustness

Full Validation + Error Handling

Symbol, side, type, quantity, and price all validated before hitting the API. BinanceAPIError, network errors, and input errors all caught and surfaced with clear, actionable messages.

Usage
CLI Commands
Run from inside the trading_bot/ directory
market_order.sh
# MARKET BUY — executes at best price instantly
python cli.py \
  --symbol   BTCUSDT \
  --side     BUY \
  --type     MARKET \
  --quantity 0.001

# Output:
==================================================
  ORDER REQUEST SUMMARY
==================================================
  Symbol   : BTCUSDT  |  Side : BUY
  Type     : MARKET   |  Qty  : 0.001
==================================================
  [OK] Order placed successfully
  Order ID : 22225283684
  Status   : NEW
limit_order.sh
# LIMIT SELL — rests on book until price reached
python cli.py \
  --symbol   BTCUSDT \
  --side     SELL \
  --type     LIMIT \
  --quantity 0.001 \
  --price    100000

# LIMIT_IOC BUY — fill or cancel (bonus type)
python cli.py \
  --symbol   BTCUSDT \
  --side     BUY \
  --type     LIMIT_IOC \
  --quantity 0.001 \
  --price    50000

# Check account balance
python cli.py --check-balance

Get Started
Quick Setup
Up and running in under 2 minutes. Free testnet API keys, no real money required.
1

Install Dependencies

Clone or unzip the project, then install the two required packages:

cd trading_bot
pip install -r requirements.txt
2

Get Testnet API Keys

Visit testnet.binancefuture.com, log in with GitHub, click avatar → API Key → Generate Key, and copy both values.

3

Configure & Trade

Copy .env.example to .env, paste your keys, and run your first order:

BINANCE_API_KEY=your_key
BINANCE_API_SECRET=your_secret
