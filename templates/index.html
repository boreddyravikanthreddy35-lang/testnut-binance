Binance Futures Testnet Credentials

These keys are only stored in your browser session (sessionStorage) — never sent to any third party. Used only to sign requests directly to testnet.binancefuture.com.

How to Get Testnet API Keys
  1. Go to testnet.binancefuture.com
  2. Click "Log In with GitHub" (free GitHub account required)
  3. Click your avatar (top-right) → "API Key"
  4. Click "Generate Key"
  5. Copy both the API Key and Secret Key (shown once!)
  6. Paste them in the fields above and click Save & Connect
